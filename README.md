# Gemini Game

This is a Flutter project for integrating Google Gemini AI chat.

# Run

## 1. Clone Project

## 2. create `.env` in your local workspace

```
GEMINI_KEY=...your gemini api key...
```

## 3. install pub

```
flutter pub get
```
